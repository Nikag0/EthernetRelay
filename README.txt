Управление Ethermet-реле по протоколу UDP.

Команда UDP пртокола:

cmSetRelays – включение/отключение электромагнитных реле.
Описание: задаёт индивидуальное состояние каждого электромагнитного реле. 
Включённому состоянию соответствует значение больше 0x00, выключенному – 0x00.
Код	команды: 0x01
Данные: <relay1>, <relay2>, 0x00, 0x00
Ответ:
Команда: 0x33
Данные: <relay1>, <relay2>, 0x00, 0x00 – изменённые состояния электромагнитных реле
